# Project File Guides

This folder explains the main source files in the `project/` package.

## Included guides

- `Interface.md`: how a run starts and how the modules are wired together
- `Initialiser.md`: manual input, saved input files, unit parsing, and file loading
- `Distributions.md`: initial position, velocity, and angular-momentum sampling
- `Particles.md`: particle objects and the vectorised particle-array engine
- `Simulation.md`: the simulation loop, sampling, summaries, and graph generation
- `Graphs.md`: plot classes and output-file behaviour
- `Assets-and-Folders.md`: what the input and output folders are for

These guides are intentionally general. They describe responsibilities, data flow, and important assumptions rather than repeating every line of code.
